# Scoop Main [![Tests](https://github.com/Unity-Billal-mesloub/Main/actions/workflows/ci.yml/badge.svg)](https://github.com/Unity-Billal-mesloub/Main/actions/workflows/ci.yml) [![Excavator](https://github.com/Unity-Billal-mesloub/Main/actions/workflows/excavator.yml/badge.svg)](https://github.com/Unity-Billal-mesloub/Main/actions/workflows/excavator.yml)

Core manifests for [Scoop](https://scoop.sh), the Windows command-line installer. For manifests that fit the [Main criteria](https://github.com/Unity-Billal-mesloub/Scoop/wiki/Criteria-for-including-apps-in-the-main-bucket).

How do I install these manifests?
---------------------------------

Just do `scoop install <manifest>`. This is the default bucket for Scoop and is added by default.

How do I contribute new manifests?
----------------------------------

To make a new manifest contribution, please read the [Contributing Guide](https://github.com/Unity-Billal-mesloub/.github/blob/main/.github/CONTRIBUTING.md).
